# Перевод всех услуг на все языки

## Автоматический перевод

После деплоя на продакшн, переводы будут генерироваться автоматически при первом запросе к API услуг (`/api/services`).

## Принудительный перевод всех услуг

Для принудительного перевода всех услуг на все языки, выполните один из следующих запросов:

### Вариант 1: Через браузер или curl

```bash
curl -X POST "https://vkbouwmaster.com/api/services/translate?force=true"
```

### Вариант 2: Через админ-панель (если есть доступ)

Откройте в браузере:
```
https://vkbouwmaster.com/api/services/translate?force=true
```

### Вариант 3: Через Render.com Logs

После деплоя, переводы будут автоматически генерироваться при первом запросе к `/api/services`.

## Проверка переводов

После запуска перевода, проверьте, что все услуги переведены:

1. Откройте любую страницу услуги, например:
   - https://vkbouwmaster.com/services/hedge-trimming
   - https://vkbouwmaster.com/services/flooring-installation
   - https://vkbouwmaster.com/services/paving-stone-installation

2. Переключите язык в интерфейсе - все тексты должны переводиться автоматически.

## Технические детали

- Переводы сохраняются в persistent storage (Render Disk или локально в `public/uploads/services-data.json`)
- Исходный язык определяется автоматически на основе текста
- Используется OpenAI API для высококачественных переводов с fallback на бесплатные API
- Переводы генерируются один раз и кешируются
